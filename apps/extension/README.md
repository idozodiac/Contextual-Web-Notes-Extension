# Extension App

Chrome Extension for Contextual Web Notes.

## Development

```bash
npm run dev
```

This will:
1. Start Vite dev server with HMR
2. Watch for changes and rebuild
3. Output to `dist/` directory

## Loading in Chrome

1. Build the extension: `npm run build`
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the `dist` folder

## Icons

Place your extension icons in the `icons/` directory:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)
